Chess Puzzle API

Puzzles supplied by the good people of Lichess.org

You can find the source code of this project Here

Base Endpoint

https://chesspuzzleapi.com/api

Return a random puzzle.

https://chesspuzzleapi.com/api/puzzle

Return a random puzzle that is greater than a rating

If one rating parameter is supplied, they resulting puzzle rating will be greater than parameter.

Example:

https://chesspuzzleapi.com/api/rating/1000

Returns a puzzle with a rating greater than 1000.

Return a random puzzle that is between two ratings

If two rating parameters are supplied, they must be sucessive and low to high.

Example:

https://chesspuzzleapi.com/api/between/1000/1200

Returns a puzzle with a rating between 1000 and 1200.

Return a random puzzle with theme

Example:

https://chesspuzzleapi.com/api/theme/endgame

Returns a puzzle with endgame theme.

Valid Themes(Case Sensitive)

To see list of valid themes see this Lichess Repo

Nested Queries

Combine rating filter along with a theme filter. Rating parameters must be sucessive and from low to high. Theme and Rating parameter can be in any order.

Example:

https://chesspuzzleapi.com/api/rating/1200/theme/advantage

Returns a puzzle with a rating over 1200 and a theme of advantage. Example:

https://chesspuzzleapi.com/api/theme/middlegame/rating/1700

Returns a puzzle with a theme of middlegame and rating over 1700. Example:

https://chesspuzzleapi.com/api/between/1000/1200/theme/endgame

Returns a puzzle with a rating between 1000 and 1200 and a theme of endgame. Example:

https://chesspuzzleapi.com/api/theme/opening/between/1000/1200

Returns a puzzle with a rating between 1000 and 1200 and a theme of opening.
